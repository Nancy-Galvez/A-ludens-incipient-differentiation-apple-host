# A-ludens-incipient-differentiation-apple-host
This repository contains scripts for population genomic analyses of the *Anastrepha ludens*, a polyphagous pest. Recently, this species has expanded from lowland tropical regions into highland temperate environments (>1,800 m) attacking commercial apples (*Malus domestica*). Using SNPs from GBS data (261 flies, 30 localities, 10 host plants).
We investigated whether apple use reflects host-driven genomic divergence or broader geographic processes. Results indicate weak genomic differentiation, with spatial structure (isolation by distance, latitude) explaining more variance than host association. Apple colonization appears to be in an early, polygenic stage without strong host-driven divergence.
**Contents:**  
- `data snp/`: SNP genotypes (two panels: host-based and locality-based)  
- `scripts/`: Workflows for PCA, PERMANOVA, Mantel tests, spatial–environmental partitioning, and genome scans  
- `results/`: Outlier detection, annotations (chemosensation, stress response, etc.)
